# Score Sync
## Description
A simple and efficient web-based application to track student grades, manage subjects, and visualize academic performance.

## Features
● Add Student grade
● Edit Student
● Delete Student
● Check status pass/fail
● view Students

## Tech Stack
● React
● Tailwindcss
● MaterialUI
● Localstorage# React + Vite
